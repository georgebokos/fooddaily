# ΜΕΡΟΣ Α — GitHub (ανέβασμα αρχείων)

## Βήμα 1: Δημιουργία Repository
1. Μπες στο **github.com** (είσαι ήδη συνδεδεμένος)
2. Πάνω δεξιά κλικ στο **+** → **New repository**
3. Repository name: `fooddaily`
4. Επίλεξε **Public**
5. Κλικ **Create repository**

## Βήμα 2: Ανέβασμα αρχείων
1. Στη νέα σελίδα κλικ **uploading an existing file** (μπλε link στη μέση)
2. Άνοιξε τον φάκελο που αποσυμπίεσες
3. **Επίλεξε ΟΛΑ τα αρχεία** (index.html, manifest.json, service-worker.js, ΟΛΑ τα icon-*.png)
   - Τράβα τα με το ποντίκι μέσα στο παράθυρο του GitHub
4. Κάτω κλικ **Commit changes**

## Βήμα 3: Ενεργοποίηση GitHub Pages
1. Στο repository κλικ **Settings** (πάνω δεξιά)
2. Αριστερά κλικ **Pages**
3. Στο "Branch" επίλεξε **main** → **Save**
4. Περίμενε 1-2 λεπτά
5. ✅ Η εφαρμογή σου είναι ΖΩΝΤΑΝΗ στο:
   **`https://ΤΟ_USERNAME_ΣΟΥ.github.io/fooddaily`**

## Βήμα 4: Δοκίμασέ την!
Άνοιξε το link στο κινητό σου → πρέπει να ανοίγει η εφαρμογή κανονικά.

# ΜΕΡΟΣ Β — PWABuilder (δημιουργία APK)

## Βήμα 5: Μετατροπή σε Android app
1. Μπες στο **pwabuilder.com**
2. Στο πεδίο βάλε το GitHub Pages link σου
3. Κλικ **Start**
4. Θα δεις score (πρέπει να είναι πράσινο/καλό)
5. Κλικ **Package For Stores**
6. Επίλεξε **Android** → **Generate Package**

## Βήμα 6: Ρυθμίσεις πακέτου
- **Package ID**: `gr.fooddaily.app`
- **App name**: `FoodDaily`
- **Launcher name**: `FoodDaily`
- Άσε τα υπόλοιπα όπως είναι
- Κλικ **Download**

## Βήμα 7: Τι κατέβασες
Μέσα στο ZIP θα βρεις:
- `fooddaily.aab` ← **ΑΥΤΟ ανεβάζεις στο Play Store**
- `signing.keystore` ← **ΦΥΛΑΞΕ ΤΟ ΚΑΛΑ!** (το χρειάζεσαι για κάθε update)
- `signing-key-info.txt` ← κωδικοί keystore — **ΚΡΑΤΑ ΤΟΥΣ**

⚠️ ΠΡΟΣΟΧΗ: Αν χάσεις το keystore, ΔΕΝ μπορείς να κάνεις update ποτέ!
Ανέβασέ το σε Google Drive ή κάπου ασφαλές.

# ΜΕΡΟΣ Γ — Google Play Console (δημοσίευση)

## Βήμα 8: Δημιουργία εφαρμογής
1. Μπες στο **play.google.com/console** (μετά την έγκριση των $25)
2. Κλικ **Create app**
3. Συμπλήρωσε:
   - App name: **FoodDaily**
   - Default language: **Greek (Ελληνικά)**
   - App or game: **App**
   - Free or paid: **Free**
4. Τσέκαρε τις δηλώσεις → **Create app**

## Βήμα 9: Συμπλήρωση στοιχείων (αριστερό μενού)

### "Store listing" (Παρουσίαση)
- **App name**: FoodDaily
- **Short description** (αντίγραψε):
  `Μεσογειακές συνταγές, καθημερινές προτάσεις & έξυπνο ψυγείο`
- **Full description** (αντίγραψε από κάτω ⬇️)
- **App icon**: ανέβασε το `icon-512.png`
- **Feature graphic**: 1024×500 px (φτιάξε στο Canva — δες παρακάτω)
- **Phone screenshots**: τράβα 2-4 από την εφαρμογή στο κινητό

### "App content" (υποχρεωτικά ερωτηματολόγια)
- Privacy policy: χρειάζεσαι link (δες ΣΗΜΕΙΩΣΗ παρακάτω)
- Ads: **No** (δεν έχει διαφημίσεις)
- Content rating: συμπλήρωσε το ερωτηματολόγιο → "Everyone"
- Target audience: 13+
- Data safety: δηλώνεις ότι δεν συλλέγεις δεδομένα

## Βήμα 10: Ανέβασμα του app
1. Αριστερά → **Production** → **Create new release**
2. Κλικ **Upload** → ανέβασε το `fooddaily.aab`
3. Release name: `1.0`
4. Release notes: `Πρώτη έκδοση — 35+ συνταγές, εορτολόγιο, έξυπνο ψυγείο`
5. Κλικ **Next** → **Save**

## Βήμα 11: Δημοσίευση
1. Κλικ **Review release**
2. Αν όλα είναι πράσινα → **Start rollout to Production**
3. ✅ Σε 1-7 μέρες η εφαρμογή είναι στο Play Store!
